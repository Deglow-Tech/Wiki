Objetivos: Medir el porcentaje de saturación de oxígeno de la hemoglobina (SaO2) en sangre y la frecuencia cardiaca de forma no invasiva.

Especificaciones técnicas:
* Voltaje de Operación: 5V DC
* Protocolo de comunicación: I2C (compatible con 5v y 3.3v)
* Led rojo de 660nm
* Led infrarrojo de 880nm
* Filtro de luz entre 50 y 60Hz
* Dimensiones: 14mm x 17mm
* Consumo: menor a 1mW
* Velocidad de muestreo: 50 a 3200 mediciones por segundo
* Memoria con cola FIFO de 32 niveles.

Pines:
* GND: Se conecta a GND.
* VCC: Se conecta a 5V.
* SCL: I2C Clock
* SDA: I2C Data
* INT: Interrupción, se encuentra desactivada cuando se conecta a GND.
